# ActionBroadcaster

A Sponge minecraft server plugin to create automated messages 
that will be printed into the action chat slot.

Tested against: 
	
	Sponge: sponge-1.8-1499-2.1DEV-532
	
	Minecraft (Client): 1.8.8
